### 本书讲了什么

#### 为什么叫做代码大全
code complete 翻译认为这个其实是 代码完成 的意思, 为了纪念第一版, 所以名字没有变

这本书讲的正是为了达到 编码完成 这一个重要的里程碑所必须的软件构建技术, 确切的说, 如何编写高质量的代码.

> 首先为人编写代码, 其次才是为机器编写, 代码主要是供人阅读的.
